# Cripto Monitor

Script Python per monitorare i prezzi delle criptovalute usando l'API di CoinGecko.
Salva i dati in un file CSV e invia un'email di alert se il prezzo scende sotto una soglia configurata.

## Requisiti

- Python 3.8+
- Librerie: `requests`, `pandas`

Installazione:
```bash
pip install -r requirements.txt
