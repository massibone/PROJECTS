# Giardino Intelligente API

API per la gestione di un giardino intelligente tramite sensori e attuatori IoT.

## Endpoint

### Sensori
- `GET /sensori` - Lista tutti i sensori
- `GET /sensori/<id>` - Dettagli di un sensore
- `POST /sensori` - Aggiungi un sensore
- `PUT /sensori/<id>` - Aggiorna un sensore
- `DELETE /sensori/<id>` - Elimina un sensore

### Attuatori
- `GET /attuatori` - Lista tutti gli attuatori
- `GET /attuatori/<id>` - Dettagli di un attuatore
- `POST /attuatori` - Aggiungi un attuatore
- `PUT /attuatori/<id>` - Aggiorna un attuatore
- `DELETE /attuatori/<id>` - Elimina un attuatore
